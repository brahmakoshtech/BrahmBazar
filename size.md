# BRAHMAKOSH - Recommended Image Sizes

To maintain the premium aesthetic and high performance of the BRAHMAKOSH store, please follow these recommended image dimensions and formats.

## 1. Brand Identity
| Asset Type | Recommended Size | Aspect Ratio | Format | Usage |
| :--- | :--- | :--- | :--- | :--- |
| **Main Logo** | 400x400 px | 1:1 | PNG (Transparent) | Navbar & Branding |
| **Favicon** | 32x32 px | 1:1 | ICO / PNG | Browser Tab |

## 2. Marketing Banners
| Asset Type | Recommended Size | Aspect Ratio | Usage |
| :--- | :--- | :--- | :--- |
| **Home Hero Banner** | 1920x850 px | ~21:9 | Main Homepage Slider |
| **Secondary Banners** | 1200x400 px | 3:1 | Promotional sections |
| **Mobile Banners** | 800x1000 px | 4:5 | Mobile-specific optimization |

## 3. Product & Catalog
| Asset Type | Recommended Size | Aspect Ratio | Format | Usage |
| :--- | :--- | :--- | :--- | :--- |
| **Product Image** | 1000x1000 px | 1:1 (Square) | JPG / WebP | Main product display |
| **Category Card** | 600x800 px | 3:4 | JPG / WebP | Category Showcase |
| **Secondary Image** | 800x800 px | 1:1 | JPG / WebP | Thumbnail/Hover gallery |

## 4. Brand & Content
| Asset Type | Recommended Size | Aspect Ratio | Usage |
| :--- | :--- | :--- | :--- |
| **Brand Story** | 800x800 px | 1:1 | "Read Our Story" section |
| **About Us / Hero** | 1200x600 px | 2:1 | Informational pages |

*   **Use WebP Format**: Always prefer `.webp` over `.jpg` or `.png` for significantly faster loading speeds without losing quality.
*   **Compression**: Use tools like [TinyPNG](https://tinypng.com) or [Squoosh](https://squoosh.app) to compress images before uploading.
